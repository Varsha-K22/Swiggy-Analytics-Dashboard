# 🍔 Swiggy Data Analysis Dashboard

## 📌 Project Overview
This project is an interactive **Swiggy Food Delivery Dashboard** built using **Power BI / Excel** to analyze food ordering patterns, revenue performance, customer ratings, and restaurant insights across different states and cities.

The dashboard provides business insights through KPIs, charts, slicers, and geographical analysis.

---

## 🎯 Objectives
- Analyze food delivery performance using Swiggy dataset
- Track revenue, orders, ratings, and restaurant performance
- Identify top-performing dishes, categories, and cities
- Study daily and weekly ordering trends
- Compare **Veg vs Non-Veg** food preferences

---

## 📊 KPIs Used
- **Total Orders**
- **Total Revenue**
- **Average Order Value (AOV)**
- **Average Rating**
- **Total Restaurants**

---

## 📈 Dashboard Features

### 🔹 Interactive Filters / Slicers
- State
- City
- Restaurant Name
- Category
- Food Type (Veg / Non-Veg)

### 🔹 Visualizations
- **Order Trend Analysis**
- **Daily Trend Analysis**
- **Weekly Trend Analysis**
- **Top 10 Dishes**
- **Category Performance**
- **City Performance**
- **Rating Analysis**
- **Map Visualization – Sales by State**
- **Veg vs Non-Veg Analysis**

Additional calculated columns were created using formulas/DAX:

- Week Number
